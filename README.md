# Tweetify

Tweetify is a twitter-like social media application to socialize and share tweets with other users.

## Tech Stack

### App

- JavaScript
- React Native
- Expo
- Apollo Client
- Kitten UI

### Server

- JavaScript
- Express.js
- Apollo Server
- Redis
- MongoDB
